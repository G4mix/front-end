# Front-end Gamix

Bem-vindo ao repositório do Front-end da plataforma Gamix! Este repositório abriga o código-fonte e os recursos relacionados à interface do usuário da plataforma Gamix, onde conectamos amantes de jogos, desenvolvedores e entusiastas para colaborar na criação de jogos incríveis.

## Sobre a Plataforma Gamix

A plataforma Gamix é um ambiente colaborativo que une pessoas com paixão por jogos, oferecendo um espaço para desenvolvedores, designers, sound designers e entusiastas de jogos se unirem para criar projetos inovadores. Este repositório contém a implementação do Front-end da plataforma, que inclui páginas de registro, login, criação de projetos, feed de atividades, descoberta de projetos e muito mais.

## Tecnologias Utilizadas

- Linguagem de Programação: TypeScript
- Framework: Next.js
- Estilização: CSS Modules
- Padronização de Código: EsLint
- Testes: Cypress
